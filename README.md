# 🚗 AI-Based Autonomous Navigation System

## 📌 Project Overview
This project demonstrates an AI-based autonomous navigation system using the A* path planning algorithm and a virtual simulation environment built with Pygame.

The system simulates how a robot or vehicle navigates from a start point to a destination while avoiding obstacles.

---

## 🎯 Problem Statement
Autonomous navigation is a key challenge in robotics and self-driving systems. The goal is to enable a machine to:
- Understand its environment
- Avoid obstacles
- Plan the shortest path
- Reach a target efficiently

---

## 🧠 Solution Approach
This project uses:
- Grid-based environment simulation
- A* Algorithm for path planning
- Real-time movement simulation using Pygame

---

## ⚙️ Tech Stack
- Python
- Pygame
- NumPy
- A* Algorithm
